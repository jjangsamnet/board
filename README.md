# 공유보드 (ShareBoard)

패들렛(Padlet)과 유사한 실시간 웹 공유 보드입니다.

## 주요 기능

- 텍스트/이미지/링크/동영상 포스트 작성
- 클립보드 붙여넣기 (Ctrl+V) 지원
- 이모지 반응 (👍 ❤️ ⭐ 😄)
- 댓글 시스템
- 3가지 레이아웃 (벽/격자/스트림)
- 6가지 배경 테마
- 관리자 설정 (댓글/반응 차단·허용)
- 실시간 커서 표시
- 접속자 수 실시간 표시

## 기술 스택

- React 19 + TypeScript
- Tailwind CSS + shadcn/ui
- Vite

## 실행 방법

```bash
npm install
npm run dev
```

## 배포

GitHub Pages로 자동 배포됩니다.
`main` 브랜치에 push하면 GitHub Actions가 자동으로 빌드 및 배포합니다.
